操作 / Controls

  • W A S D 移动 / Move
  • 鼠标移动 环视 / Look
  • Space 跳跃（行走模式）/ Jump (walk mode)
  • F 切换飞行 / Toggle fly
  • Shift 下落（飞行）/ Down (fly)
  • 1-6 选择方块 / Select block
  • 左键 破坏方块 / Break
  • 右键 放置方块 / Place
  • Esc 释放鼠标 / Unlock

神话地标 / Mythic Landmarks

  • 世界树 / World Tree：出生点附近的巨树与符文祭坛
  • 浮空神殿 / Sky Temple：东南方向的漂浮岛屿与传送环
  • 龙骨峡谷 / Dragonbone Ravine：西北方向的骨架脊梁与肋骨拱门
  • 水晶尖塔 / Crystal Spires：西南方向的符文尖塔群
  • 古代遗迹 / Ancient Ruins：南方方向的断墙、拱门与柱廊
